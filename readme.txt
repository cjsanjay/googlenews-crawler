#Google News Crawler and Data Extractor

1. Use googlecrawler.py to generate the listing of news articles.
    -Modify variable word in script with the text you want to search with
    -Usage: python googlenewscrawler.py <start year> <end year> <num_pages>

2. use extractnews.py with data file generated to extract news articles.
    -Usage: python extractnew.py <data_file_location>
